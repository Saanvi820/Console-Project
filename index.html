Project On Console

How to set up Visual Studio Code: First, you need to download Visual Studio Code on your computer. Make sure you do the setup. Next, open up the software. After that, click "Explorer", and add in the Live Server. Finally, open up a folder, and start coding!

What is Console: The function console.log() is used to print any type of variables defined before it or just to print a message that the user needs to see. To open the console, put in a console.log in the code, and when you preview, right click on the screen. Click "Inspect" and click on the console. That way you will be able to see everything you put in the console.log
